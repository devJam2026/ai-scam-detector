# Cost & Infrastructure Comparison (Updated)

Overview

- A concise comparison of costs and infrastructure for the approaches included in this repo.

Summary

- `openai_llm`: low infra, medium latency/cost, external dependency
- `ml_classifier`: low infra, low latency, requires labeled data
- `local_llm`: medium-high infra, better privacy, higher ops overhead
- `hybrid`: high complexity, useful for evaluation and learning

Practical notes

- This repo is educational; figures are illustrative.
- Use OpenAI for quick prototyping; use ML or local LLMs for privacy or cost reasons depending on scale.
